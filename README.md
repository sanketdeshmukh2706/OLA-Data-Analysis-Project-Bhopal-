# 🚖 OLA Data Analysis Project (Bhopal)

## 📌 Project Overview
This project focuses on analyzing OLA ride booking data for **Bhopal city** to extract meaningful insights and improve business decision-making.

The project covers complete data analysis lifecycle including:
- Data Generation
- Data Cleaning
- Data Analysis using SQL
- Data Visualization using Power BI

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel (Data Cleaning & Preparation)
- SQL (Data Analysis)
- Power BI (Dashboard Creation & Visualization)

---

## 📂 Dataset Information
The dataset contains **1 lakh+ rows** of ride booking data for one month.

### 📊 Key Columns:
- Date & Time
- Booking ID
- Booking Status (Success / Cancelled / Incomplete)
- Customer ID
- Vehicle Type (Auto, Mini, Sedan, Bike, SUV, etc.)
- Pickup & Drop Location
- Ride Distance
- Booking Value
- Cancellation Reasons (Customer & Driver)
- Driver Ratings
- Customer Ratings
- Payment Method

---

## 🔄 Project Workflow

### 1️⃣ Data Generation & Collection
- Created a large dataset with realistic ride booking scenarios
- Ensured proper distribution of:
  - Successful bookings (~62%)
  - Customer cancellations (<7%)
  - Driver cancellations (<18%)

---

### 2️⃣ Data Cleaning (Excel)
- Removed duplicates
- Handled missing values
- Standardized column names
- Ensured correct data types

---

### 3️⃣ Data Analysis (SQL)
Performed various SQL queries to extract insights:

- Retrieved successful bookings
- Calculated average ride distance per vehicle type
- Counted cancelled rides (customer & driver)
- Identified top 5 customers by number of rides
- Analyzed driver ratings (max & min)
- Extracted rides with UPI payments
- Calculated total booking value
- Analyzed incomplete rides with reasons

---

### 4️⃣ Data Visualization (Power BI)

Created an interactive dashboard with the following insights:

#### 📈 Overall Analysis
- Ride Volume Over Time
- Booking Status Breakdown

#### 🚗 Vehicle Analysis
- Top Vehicle Types by Ride Distance

#### 💰 Revenue Analysis
- Revenue by Payment Method
- Top Customers by Booking Value

#### ❌ Cancellation Analysis
- Customer Cancellation Reasons
- Driver Cancellation Reasons

#### ⭐ Ratings Analysis
- Driver Ratings Distribution
- Customer Ratings Distribution

---

## 📊 Key Insights

- Most rides are successfully completed (~62%)
- Peak ride demand observed on weekends
- Higher revenue generated during weekends
- Certain vehicle types contribute more to total distance
- Driver-related cancellations are higher than customer cancellations
- Ratings analysis helps identify service quality

---

## 🎯 Conclusion

This project helps in:
- Understanding customer behavior
- Improving ride success rate
- Reducing cancellations
- Enhancing customer satisfaction
- Supporting data-driven business decisions
